description=[h3]Made for Multiplayer & Singleplayer.[/h3]
description=
description=[h2]Features[/h2]
description=- Set the world date.
description=- Set the world time.
description=- Synchronize to the system time.
description=- Do all that from a UI.
description=
description=[h2]Sandbox Options[/h2]
description=- Disable the mod without restarting.
description=- Auto-synchronize with system-time.
description=- Set your own timezone.
description=
description=[h2]Real-time synchronization[/h2]
description=Real-time sync is optional, the mod allow you to set any date and time for any Day length.
description=Make sure to set the [b]Day length to Real-time[/b] for the system time sync to work.
description=World time will sync when server boot and also when you change the timezone.
